LogicApp

El aplicativo "logicApp" es una herramienta de software desarrollada específicamente para facilitar el aprendizaje y aplicación de los conceptos de tablas de verdad y conjuntos de forma práctica y eficiente. El aplicativo consta de dos módulos principales: uno para tablas de verdad y otro para conjuntos.
